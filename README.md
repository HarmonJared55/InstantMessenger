#CS240 Final Project 

##Overview
This project was designed to show what I have learned in csci24000. This program is an instant messenger server and client. This project demostrates the use of OOP, Data Structures, and abstraction.

##Running the program 
The program has two seperate programs that has to be run. The server, and the client. 

###Running the server

	1. To run the server, run the following commands into a bash terminal
	```
	make clean
	make 
	make runserver
	``` 

	The commands will first clean all the class files from the directory
	compile the program 
	run the server

###Running the client
	
	2. To run the Client, enter the following commands into a bash terminal
	```
	make runclient
	```
